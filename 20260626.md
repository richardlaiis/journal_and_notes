## Solution
1. 將原本的陣列轉換成這樣的形式：如果=target，則為1，不為target即為-1
2. 維護前綴和，注意到 `pref[j]>pref[i]` for `j>=i` 為滿足條件的subarray
3. 我們可以維護另一個陣列，代表**當前prefix sum** 出現的次數
4. 對於當前的 prefix sum，我們就可以去找到目前為止 prefix sum 嚴格小於現在的總次數，當然這一項數值也可以再利用前綴和來降低複雜度
## Code

```cpp
class Solution {
    public:
        long long countMajoritySubarrays(vector < int > & nums, int target) {
            int n = nums.size();
            vector < int > pref(n * 2 + 1, 0);
            pref[n] = 1; // initially prefix sum = 0
            int cnt = n; // 0
            long long res = 0, presum = 0;
            for (int i = 0; i < n; i++) {
                if (nums[i] == target) {
                    presum += pref[cnt];
                    cnt++;
                    pref[cnt]++;
                } else {
                    cnt--;
                    presum -= pref[cnt];
                    pref[cnt]++;
                }
                res += presum;
            }
            return res;
        }
};
```
