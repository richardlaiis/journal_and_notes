## LeetCode 3020. Find the Maximum Number of Elements in Subset
### Problem Description
https://leetcode.com/problems/find-the-maximum-number-of-elements-in-subset/
### Approach
首先考慮特殊情況，也就是 1 的 case，很容易知道最小值為1的subset只會有1，而且長度必為奇數。

接著我們可以善用 hash table，存每個數出現的 frequency。接著去traverse這個table，對於每一個key，我們都把他當成subset中最小的值去暴力做 (這樣做會是好的因為 $\log_2{\log_2{(10^9)}}\approx 4.9$，序列最長的長度不會超過9)

基本上就跟程式碼所示一樣，迴圈停止的條件就是當某個 key query 出來的值是 1 或 0，如果是 0，那我們就把 key/2 刪掉其中一項，如果是 1，代表這個 key 是中間項，針對 len 的更新需要特別留意這兩種情況。
### Code
```cpp
class Solution {
    public: int maximumLength(vector < int > & nums) {
        unordered_map < long long, int > freq;
        for (auto x: nums) freq[x]++;

        // we can only select odd number of 1's
        int res = 1;
        if (freq.count(1)) {
            res = freq[1];
            if ((res & 1) == 0) res--;
        }

        freq.erase(1); // to prevent forever loop

        for (auto x: freq) {
            long long temp = x.first;
            int len = 0;
            for (; freq.contains(temp) && freq[temp] > 1; temp *= temp) {
                len += 2;
            }

            if (freq.contains(temp)) {
                len++;
            } else {
                len--;
            }

            res = max(res, len);
        }

        return res;
    }
};
```
